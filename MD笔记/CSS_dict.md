# 这是我的 CSS 字典(询问 AI 后整合而成)

## CSS常用词汇表——摆脱盲目尝试
---

### 一、布局类：控制元素位置和排列（最核心）

| 属性 | 常用值 | 作用与场景 |
| :--- | :--- | :--- |
| **`display`** | `block` | 默认，独占一行（如 `<div>`、`<p>`） |
| | `inline` | 行内，不换行（如 `<span>`、`<a>`） |
| | **`flex`** | **弹性布局，最常用的一维布局** |
| | **`grid`** | **网格布局，最强大的二维布局** |
| **`justify-content`** (flex/grid) | `flex-start` | 主轴开始对齐（左/上） |
| | **`center`** | **主轴居中对齐** |
| | `space-between` | 两端对齐，中间平均分布 |
| | `space-around` | 每个项目两侧间隔相等 |
| **`align-items`** (flex/grid) | `stretch` | 交叉轴拉伸填满（默认） |
| | **`center`** | **交叉轴居中对齐** |
| | `flex-start` | 交叉轴开始对齐（上/左） |
| **`flex-direction`** | `row` | 主轴为水平方向（默认） |
| | `column` | **主轴为垂直方向**（常用做垂直排列） |
| **`position`** | `static` | 默认，正常文档流 |
| | **`relative`** | **相对定位**（微调位置，不脱离文档流） |
| | **`absolute`** | **绝对定位**（相对于已定位的父元素） |
| | `fixed` | 固定定位（相对于浏览器窗口） |

---

### 二、盒子模型类：控制大小、间距和边框

| 属性 | 常用值（单位） | 作用与场景 |
| :--- | :--- | :--- |
| **`width`/`height`** | `200px` | 固定像素值 |
| | `50%` | 相对于父元素的百分比 |
| | `auto` | 自动计算（默认） |
| **`margin`** | `10px` | 四个方向外间距 |
| | `10px 20px` | 上下`10px`，左右`20px` |
| | `0 auto` | **上下0，左右自动 → 实现水平居中** |
| **`padding`** | 同 `margin` | 四个方向**内边距** |
| **`border`** | `1px solid #ccc` | **1像素、实线、灰色的边框**（最常用） |
| | `2px dashed #ff0000` | 2像素、虚线、红色边框 |
| **`border-radius`** | `8px` | 圆角大小 |
| | `50%` | **设为圆形** |
| **`box-sizing`** | `content-box` | 传统模型（width只含内容） |
| | **`border-box`** | **现代模型（width包含边框和内边距）**（建议全局设置） |

---

### 三、文字排版类

| 属性 | 常用值 | 作用与场景 |
| :--- | :--- | :--- |
| **`font-size`** | `16px` | 字体大小（基础大小） |
| | `1.2rem` | 相对于根元素的字体大小（推荐） |
| | `14px` | 小号字体 |
| **`font-weight`** | `normal` | 正常粗细 |
| | **`bold`** | **加粗** |
| **`color`** | `#333` | 深灰色文字（最常用，比纯黑柔和） |
| | `#ff0000` / `red` | 红色 |
| | `#007bff` | 一种标准的蓝色（常用于链接） |
| **`text-align`** | `left` | 左对齐（默认） |
| | **`center`** | **居中对齐** |
| | `right` | 右对齐 |
| **`line-height`** | `1.5` | 行高为字体的1.5倍（阅读舒适） |
| | `24px` | 固定行高 |
| **`text-decoration`** | `none` | **去除下划线**（常用于链接） |
| | `underline` | 添加下划线 |

---

### 四、颜色与背景类

| 属性 | 常用值 | 作用与场景 |
| :--- | :--- | :--- |
| **`background-color`** | `white` | 白色背景 |
| | `#f5f5f5` | 浅灰色背景（非常常用） |
| | `rgba(0, 0, 0, 0.5)` | 黑色，50%透明度 |
| **`background-image`** | `url("图片地址.jpg")` | 设置背景图片 |
| **`background-size`** | `cover` | 覆盖整个区域（可能裁剪图片） |
| | `contain` | 完整包含图片（可能留白） |

---

### 五、其他高频实用属性

| 属性 | 常用值 | 作用与场景 |
| :--- | :--- | :--- |
| **`cursor`** | `pointer` | 鼠标悬停时变为**手型**（表示可点击） |
| **`overflow`** | `visible` | 内容溢出时显示（默认） |
| | `hidden` | **溢出部分隐藏** |
| | `auto` | 溢出时出现滚动条 |
| **`opacity`** | `1` | 完全不透明 |
| | `0.5` | 50%透明度 |
| **`transition`** | `all 0.3s ease` | **所有属性变化时，在0.3秒内平滑过渡**（创造动画感） |
