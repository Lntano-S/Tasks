# HTML
> 全称是 `Hypertext Markup Language(超文本标记语言)`
 
HTML用一系列预定义的标签，来告诉浏览器：“这里是一级标题”、“这里是一张图片”、“这里是一个链接”等内容

## 一、HTML文件的基本结构

```HTML
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个网页，我成功了！</p>
</body>
</html>
```

1. `<!DOCTYPE html>`：**文档类型声明**。这是告诉浏览器：“请用最新的HTML标准来渲染这个页面”
2. `<html>`：**根标签**。像一个大容器，所有其他标签都必须放在这个标签里面，它表示这是一个HTML文档。
3. `<head>`：**头部**。这里面放的是“给浏览器看的信息”，而不是在网页上直接显示出来的内容。比如页面的标题、字符编码等。
4. `<title>`：**标题**。浏览器标签上显示的文本
5. `<body>`：**身体**。所有在网页上能看到的内容都必须放在这个标签里面。

## 二、常用文本标签

### 1. **标题标签**：`<h1>这是一级标题</h1>`
和 *Markdown* 类似，它也有六级标题
> *注意*：按照层级顺序使用标题，不要跳级（比如在 `<h1>` 后面直接跟一个 `<h3>`）

### 2. **段落标签**：`<p>这是一个段落</p>`

更改文本样式：
```HTML
    <p>
        <strong>加粗</strong>、<em>倾斜</em>、<u>下划线</u>、<s>删除线</s> 
    </p>
```
### 3. **列表标签**：`<li>这是一个项目</li>`

- 无序列表 `<ul>` 
例如：
```HTML
    <p>我需要到学生服务中心购买的物品：</p>
    <ul>
        <li>酸奶</li>
        <li>泡面</li>
        <li>柚子</li>
    </ul>
```

- 有序列表 `<ol>`
例如：
```HTML  
    <p>如何观看<i>《罗小黑战记2》</i>？</p>
    <ol>
        <li>充值一个B站大会员</li>
        <li>点击搜索栏，搜索《罗小黑战记2》</li>
        <li>点击全屏，开始观影吧！</li>
    </ol>
```

- 嵌套列表：顾名思义，有序列表与无序列表可以互相套娃
```HTML
<ul>
    <li><p>第一个有序列表</p><ol>
        <li></li>
        <li></li>
        <li></li>        
    </ol></li>
    <li><p>第二个有序列表</p><ol>
        <li></li>
        <li></li>
        <li></li>
    </ol></li>
</ul>
```

## 三、HTML属性

- 基本语法：`<开始标签 属性名="属性值">`
- 每个HTML元素可以具有不同属性
- 属性名称不区分大小写，属性值对大小写敏感：
```HTML
<img src="LXH.jpg" alt="">
<img SRC="LXH.jpg" alt="">
<img src="LXH.JPG" alt="">
```
前两者相同，第三个与前两个不同

1. **链接标签**：`<a href="URL">"用户端显示的文本内容"</a>` 

    `href`属性用来连接 **希望用户点击后跳转到的地方**，可以是绝对路径（URL），也可以是相对路径（网站的其他页面）
```HTML
    <p>
        你可以通过 <a href="https://luoxiaohei.huijiwiki.com/wiki/">罗小黑百科</a>来了解更多关于罗小黑的内容
    </p>
```
1. **图片标签**：`<img src="路径" alt="未加载出来是显示的文字" width="调整图片大小">`
