# CSS
> 全称是 **Cascading Style Sheets** 的缩写，中文意思是“层叠样式表”。
> 用于定义网页样式和布局的样式表语言，包括元素的颜色、大小、间距、边框、背景等

**核心思想**：将内容（HTML）与表现（CSS）分离。HTML 是毛胚房，CSS 是装修。你可以只修改一个 CSS 文件，就能改变整个网站的外观，而无需触碰任何 HTML 代码。

## 一、CSS语法

基本结构：
```CSS
选择器{
    属性1:属性值1；
    属性2:属性值2；
}
```
1. 选择器的声明中可以写无数条属性
2. 声明的每一行属性，都要以 `;` 结尾
3. 声明中的所有属性和值都是以**键值对**的形式出现

示例：
```CSS
/*这是一个 <p> 标签选择器*/
p {
    color: blue;
    font-size: 16px;
}
```

## 二、如何在 HTML 中使用 CSS？

**最优解**：使用外部样式表，操作如下：
1. 创建一个新的文件，以 `.css` 结尾（例如 `style.css`）。
2. 将所有的 CSS 代码写在这个文件里。
3. 在 HTML 文件的 `<head>` 部分，使用 `<link>` 标签来引入这个 CSS 文件。

**优点**：

- **完全分离**：HTML 和 CSS 在不同的文件中。
- **可重用**：同一个 `style.css` 文件可以被网站中的任意多个页面链接使用，保证了网站风格的统一。
- **易于维护**：想修改样式，只需要找到对应的 CSS 文件即可。

## 三、简单的动手实践

1. **创建 HTML 文件** (index.html)，并在 `<head>` 部分中添加 `<link rel="stylesheet" href="style.css">`，来告诉浏览器：“请加载并使用 style.css 这个文件来作为本页面的样式表”。
2. **创建 CSS 文件** (style.css)，来改变HTML文件在浏览器中的渲染效果。

例如：
- style.css
```css
h1 {
  color: blue;
}

p {
  color: gray;
  font-size: 14px; /* 设置段落的字体大小为14像素 */
}
```

> 但是问题来了，如此使用样式表，每次**选择器**会选中所有的“标题”、“段落”标签等，我们能不能精确到每个标签呢？

## 四、类选择器 和 ID 选择器

### 类选择器 (选择器：`.classname`)

“类”相当于一个用于辨别的**标签**。你可以给一个或多个 HTML 元素贴上同一个“标签”，然后专门为贴有这个标签的元素设置样式。

- **HTML 中**: 使用 `class` **属性**来给元素分配一个类名。
- **CSS 中**: 在类名前加上一个**点 (`.`)** 来选中它们。

例如：
**HTML (`index.html`)**
```HTML
<p>这是一个普通的段落</p>
<p class="important">这是一个重要段落！</p>
```

**CSS (`style.css`)**
```CSS
/* 普通段落样式 */
p {
  color: gray;
}

/* "warning"类的专属样式 */
.warning {
  color: red;
  font-size: 14px；
}
```

### ID 选择器 (选择器：`#idname`)

"ID" 相当于一个人的**身份证号**，它在一个页面上必须是**独一无二**的，一个 ID 名只能分配给**一个** HTML 元素。
- **HTML 中**: 使用 `id` **属性**来给元素分配一个唯一的 ID。
- **CSS 中**: 在 ID 名前加上一个 `#` 来选中它。

例如：
**HTML (`index.html`)**
```HTML
<h1 id="main-title">网站的主标题</h1>
```

**CSS (`style.css`)**
```CSS
#main-title {
  color: navy;
  font-size: 32px;
}
```
