# CSS
> 全称是 **Cascading Style Sheets** 的缩写，中文意思是“层叠样式表”。
> 用于定义网页样式和布局的样式表语言，包括元素的颜色、大小、间距、边框、背景等

**核心思想**：将内容（HTML）与表现（CSS）分离。HTML 是毛胚房，CSS 是装修。你可以只修改一个 CSS 文件，就能改变整个网站的外观，而无需触碰任何 HTML 代码。

## 一、CSS语法

基本结构：
```CSS
选择器{
    属性1:属性值1；
    属性2:属性值2；
}
```
1. 选择器的声明中可以写无数条属性
2. 声明的每一行属性，都要以 `;` 结尾
3. 声明中的所有属性和值都是以**键值对**的形式出现

示例：
```CSS
/*这是一个 <p> 标签选择器*/
p{
    color: blue;
    font-size: 16px;
}
```
